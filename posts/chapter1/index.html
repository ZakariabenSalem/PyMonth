Overview

No one can deny that Python is one of the most used programming languages of our time.

The vast community of python allows it to progress further and venture through many different domains.

Python is an easy to use language thanks to the easy syntax and flexibility it offers. However Python is an interpreted language, thus it treats each line on its own. It goes through the file line by line unlike C or C++ which compile the whole file and generates and output file to be executed.

Basic list operations

Here are some basic operations you can apply on lists.

people.append('Zakaria')

# Remove from list
people.remove('Mahmoud')

# Insert into position
people.insert(2, 'Omar')

# Change value
people[0] = 'Rania'

# Remove with pop
people.pop(2)

# Reverse list
people.reverse()

# Sort list
people.sort()

# Reverse sort
people.sort(reverse=True)

If/ Else

If conditions are used to decide to do something based on something being true or false


x = 21
y = 20

Simple if

Comparison Operators (==, !=, >, <, >=, <=) - Used to compare values


if x > y:
  print(f'{x} is greater than {y}')

If/else


if x > y:
  print(f'{x} is greater than {y}')
else:
  print(f'{y} is greater than {x}')

elif


if x > y:
  print(f'{x} is greater than {y}')
elif x == y:
  print(f'{x} is equal to {y}')  
else:
  print(f'{y} is greater than {x}')

Nested if


if x > 2:
  if x <= 10:
    print(f'{x} is greater than 2 and less than or equal to 10')

Logical operators

Used to combine conditional statements

  • and
  • or
  • not

and


if x > 2 and x <= 10:
    print(f'{x} is greater than 2 and less than or equal to 10')

or


if x > 2 or x <= 10:
    print(f'{x} is greater than 2 or less than or equal to 10')

not


if not(x == y):
  print(f'{x} is not equal to {y}')

Membership Operators

Membership operators are used to test if a sequence is presented in an object

  • not
  • not in

numbers = [1,2,3,4,5]

####  in
if x in numbers:
  print(x in numbers)

#### not in
if x not in numbers:
  print(x not in numbers)

Identity Operators

Compare the objects, not if they are equal, but if they are actually the same object, with the same memory location:

  • is
  • is not

# is
if x is y:
  print(x is y)

# is not
if x is not y:
  print(x is not y)

Loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

people = ['Hama', 'Hiba', 'Aziz', 'Omar']

Simple for loop

for person in people:
  print(f'Current Person: {person}')

Break

for person in people:
  if person == 'Aziz':
    break
  print(f'Current Person: {person}')

Continue

for person in people:
  if person == 'Hama':
    continue
  print(f'Current Person: {person}')

range

for i in range(len(people)):
  print(people[i])

for i in range(0, 11):
  print(f'Number: {i}')

While

while loops execute a set of statements as long as a condition is true.

count = 0
while count < 10:
  print(f'Count: {count}')
  count += 1

Functions

Definition

A function is a block of code which only runs when it is called. In Python, we do not use curly brackets, we use indentation with tabs or spaces

Create function

def sayHello(name='zak'):
    print(f'Hello {name}')

Return values

def getSum(num1, num2):
    total = num1 + num2
    return total

Lamda functions

  • A lambda function is a small anonymous function.
  • A lambda function can take any number of arguments, but can only have one expression. Very similar to JS arrow functions.

getSum = lambda num1, num2: num1 + num2

print(getSum(1, 1))
